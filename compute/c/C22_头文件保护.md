# 头文件保护

[toc]

## 例

 文件hh.h

```c
int num = 1;
```

 文件h1.h

```c
#include "hh.h"
```

文件h2.h

```c
#include "hh.h"
```

文件main.c

```c
#include "h1.h"
#include "h2.h"
int main(void){
    num = 12;
	return 0;
}
```

由于都引用了同一个头文件，导致main中交汇时重复定义了，预编译时main就会出错；

于是需要头文件保护;

## 头文件保护方法

修改文件hh.h，检查是否定义了 \_hh\_ 如果没有则define ，这样就避免预编译时h1和h2重复引用了；

```c
#ifndef _hh_  
#define _hh_ 
int num = 1;
#endif
```
