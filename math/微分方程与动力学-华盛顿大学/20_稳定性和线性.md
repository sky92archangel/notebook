## 1. 特征值的意义与稳定性分析

### 1.1 标量方程示例
$$\dot{x} + 3\dot{x} + 2x = 0$$
解的形式：
$$x(t) = c_1 e^{-t} + c_2 e^{-2t}$$

- $e^{-t}$、$e^{-2t}$ 为**稳定**（decay）项，系统才稳定。
- 若出现 $e^{at}$（$a>0$），系统发散，不稳定。

### 1.2 复特征值情形
若 $\lambda = a + ib$，则
$$e^{\lambda t} = e^{at} e^{ibt} = e^{at} \big[\cos(bt) + i\sin(bt)\big]$$
- $a \in \mathbb{R}$ 决定**稳定性**：
  - $a < 0$：衰减（stable）
  - $a = 0$：中性稳定（marginally stable）
  - $a > 0$：发散（unstable）
- $b$ 决定振荡频率（scale, unit length in complex plane）。

### 1.3 线性系统 $\dot{x} = Ax$ 的稳定性判据
- 所有特征值 $\lambda_i$ 的实部 $\text{Re}(\lambda_i) < 0$：系统**渐近稳定**。
- 若存在 $\text{Re}(\lambda_i) > 0$：系统**不稳定**。
- 若 $\text{Re}(\lambda_i) \le 0$，且零实部特征值为单根：**临界稳定**（marginally stable）。

## 2. 线性的意义

### 2.1 线性组合的意义
对 $\dot{x} = Ax$，若 $x_1(t), x_2(t)$ 是解，则任意线性组合 $c_1 x_1(t) + c_2 x_2(t)$ 仍为解，这是线性系统的核心性质。

**示例**：
设 $\dot{x} = -x$，解为 $x_1 = e^{-t}$；$\dot{x} = -2x$，解为 $x_2 = e^{-2t}$。
线性组合 $x = \alpha e^{-t} + \beta e^{-2t}$ 仍为解：
$$\dot{x} = -\alpha e^{-t} - 2\beta e^{-2t}$$
代入验证：
$$\ddot{x} = \alpha e^{-t} + 4\beta e^{-2t}$$
$$\ddot{x} + 3\dot{x} + 2x = (\alpha e^{-t} + 4\beta e^{-2t}) + 3(-\alpha e^{-t} - 2\beta e^{-2t}) + 2(\alpha e^{-t} + \beta e^{-2t}) = 0$$
成立。

### 2.2 线性与非线性映射的定义

- **线性**：$f(x+y) = f(x) + f(y)$，$f(\alpha x + \beta y) = \alpha f(x) + \beta f(y)$，满足叠加原理。
- **非线性**：$f(x) = x^2$，$f(y) = y^2$，则 $f(x+y) = (x+y)^2 = x^2 + y^2 + 2xy \neq f(x) + f(y)$，不满足叠加原理。

### 2.3 线性算子示例（Maxwell 方程）
$$A(\alpha \vec{x}_1 + \beta \vec{x}_2) = \alpha A \vec{x}_1 + \beta A \vec{x}_2$$
若 $f(x) = \frac{dx}{dt}$，则：
$$\frac{d}{dt}(\alpha x + \beta y) = \alpha \frac{dx}{dt} + \beta \frac{dy}{dt} = \alpha f(x) + \beta f(y)$$
故 $\frac{d}{dt}$ 是线性算子。

## 3. 线性系统的一般形式

线性齐次系统：
$$\dot{x} = Ax$$
其中 $A$ 为常数矩阵，解的形式为 $x(t) = e^{At} x(0)$，由矩阵指数给出。

## 4. 相图与稳定性示意

- 稳定焦点（stable focus）：特征值为共轭复根，$\text{Re}(\lambda) < 0$，轨迹向原点收敛。
- 不稳定焦点（unstable focus）：$\text{Re}(\lambda) > 0$，轨迹远离原点。
- 中心（center）：$\text{Re}(\lambda) = 0$，纯虚根，轨迹为闭合椭圆。
- 结点（node）：实特征值，同号且不等，轨迹沿特征方向收敛或发散。
