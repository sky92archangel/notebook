 

# 数值积分 Numerical Integration 完整笔记
## 一、积分定义基础
目标积分：
$$
\int_a^b f(x) dx
$$
设区间离散划分：$x_k$ 为采样点，$f_k = f(x_k)$
黎曼积分定义：
$$
\begin{aligned}
\int_a^b f(x) dx &= \lim_{N\to\infty} \sum_{k=1}^N f(a+k\Delta x)\Delta x \\
&= \lim_{N\to\infty} \sum_{k=0}^{N-1} f(x_k)\Delta x
\end{aligned}
$$

---
## 二、矩形法（Rectangle Rule）
### 1. 左端点矩形法 Left Rectangle
取区间左端点函数值作为矩形高：
$$
\int_{x_0}^{x_1} f(x)dx \approx f(x_0)\Delta x
$$
特性：计算结果通常**偏小**。

### 2. 右端点矩形法 Right Rectangle
取区间右端点函数值作为矩形高：
$$
\int_{x_0}^{x_1} f(x)dx \approx f(x_1)\Delta x
$$
特性：计算结果通常**偏大**。

### 误差阶数
单区间局部误差：$O(\Delta x^2)$
全局总误差：$O(\Delta x)$

---
## 三、梯形法 Trapezoidal Rule
用区间两端点连线构成梯形近似曲边梯形：
$$
\int_{x_0}^{x_1} f(x)dx \approx \frac{\Delta x}{2}\big(f(x_0)+f(x_1)\big)
$$
### 特性
1. 精度高于矩形法，区间分布不均时仍有较好效果；
2. 若被积函数 $f(x)$ 曲率过大，会产生明显误差；
### 误差阶数
单区间局部误差：$O(\Delta x^3)$
全局总误差：$O(\Delta x^2)$

---
## 四、辛普森法 Simpson Rule（三点二次插值）
每2个区间（3个采样点）做三次多项式插值近似曲线：
$$
\int_{x_0}^{x_2} f(x)dx = \frac{\Delta x}{3}\big(f_0 + 4f_1 + f_2\big) - \frac{\Delta x^5}{90}f^{(4)}(\xi)
$$
### 特性
1. 仅需3个点即可达到高阶精度；
2. 对分段二次曲线完全精确；
### 误差阶数
单区间局部误差：$O(\Delta x^5)$
全局总误差：$O(\Delta x^4)$

---
## 五、三种方法精度对比表
| 积分方法       | 局部单区间误差  | 全局总误差阶    | 核心特点             |
| -------------- | --------------- | --------------- | -------------------- |
| 矩形法（左右） | $O(\Delta x^2)$ | $O(\Delta x)$   | 计算最简单，精度最低 |
| 梯形法         | $O(\Delta x^3)$ | $O(\Delta x^2)$ | 线性插值，中等精度   |
| 辛普森法       | $O(\Delta x^5)$ | $O(\Delta x^4)$ | 二次插值，高阶高精度 |



 



```python
import numpy as np
from matplotlib import pyplot as plt

# ===================== 1. 参数与高精度细网格数据 =====================
# 积分上下限
a = 0
b = 10
# 细网格步长（高精度参考曲线）
dxf = 0.01
xf = np.linspace(a, b, int(b / dxf))
yf = np.sin(xf)

# 绘制原始sin(x)曲线
plt.figure(figsize=(10,4))
plt.grid(True)
plt.plot(xf, yf, label=r'$y=\sin(x)$', color='black')

# ===================== 2. 粗网格（用于矩形/梯形近似绘图） =====================
dxc = 0.5
xc = np.linspace(a, b, int(b / dxc))
edges = np.linspace(a, b, int(b / dxc) + 1, endpoint=True)
yc = np.sin(xc)

# 绘制左矩形阶梯示意（红色）
plt.stairs(yc, edges, color='r', alpha=0.4, label='Left Rectangle Approx')
plt.legend()
plt.show()

# ===================== 3. 循环实现三种数值积分方法 =====================
n = len(xc)

# 左矩形法则 Left Rectangle Rule
areal = 0
for i in range(n - 1):
    areal = areal + yc[i] * dxc
print("左矩形积分结果(循环):", areal)

# 右矩形法则 Right Rectangle Rule
area2 = 0
for i in range(n - 1):
    area2 = area2 + yc[i + 1] * dxc
print("右矩形积分结果(循环):", area2)

# 梯形法则 Trapezoidal Rule
area3 = 0
for i in range(n - 1):
    area3 = area3 + (dxc / 2) * (yc[i] + yc[i + 1])
print("梯形积分结果(循环):", area3)
print("-" * 60)

# ===================== 4. Numpy向量化快速计算（等价循环） =====================
areal_vec = sum(yc[:-1]) * dxc
area2_vec = sum(yc[1:]) * dxc
area3_trapz = np.trapz(yc, xc)
print("向量化左矩形 / 右矩形 / np.trapz(xc):", areal_vec, area2_vec, area3_trapz)

# 无x参数的np.trapz（默认均匀步长，乘dxc）
area3_trapz2 = np.trapz(yc) * dxc
print("向量化左矩形 / 右矩形 / np.trapz(yc)*dxc:", areal_vec, area2_vec, area3_trapz2)
print("-" * 60)

# ===================== 5. 细网格高精度梯形积分（逼近真值1.8391） =====================
arealf = sum(yf[:-1]) * dxf
area2f = sum(yf[1:]) * dxf
area3f = np.trapz(yf, xf)
print("粗网格左矩形 / 粗网格右矩形 / 细网格梯形高精度:", areal, area2, area3f)

area3f2 = np.trapz(yf) * dxf
print("粗网格左矩形 / 粗网格右矩形 / 细网格梯形无坐标:", areal, area2, area3f2)

# 理论解析真值 ∫₀¹⁰ sinx dx = -cos(10)+cos(0)
true_val = -np.cos(10) + np.cos(0)
print("\n理论精确积分真值:", true_val)
```
